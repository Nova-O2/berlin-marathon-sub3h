# Data

## Source

The original dataset contains 880,779 finisher records from the BMW Berlin Marathon (1999–2025), extracted from the official results archive.

## How to obtain

Download `Dataset_Berlin_Marathon_1999-2025_original.csv` from Zenodo and place it in this directory:

```
data/
└── Dataset_Berlin_Marathon_1999-2025_original.csv
```

## Reproduction pipeline

Run the notebooks in order from `notebooks/`. Each step reads from and writes to this directory:

```
1. CLEANING.ipynb
   Reads:  Dataset_Berlin_Marathon_1999-2025_original.csv (880,779 records)
   Writes: berlin_marathon_clean.parquet (834,168 records)
   Action: Harmonize gender, convert times, filter implausible values,
           remove missing splits, monotonicity check

2. COHORT_DEFINITION.ipynb
   Reads:  berlin_marathon_clean.parquet
   Writes: cohort_success.csv (7,710 runners)
           cohort_failure.csv (1,875 runners)
   Action: Apply Golden Window matching (HM 1:27:35–1:28:35),
           define Success (2:58:00–2:59:59) and Failure (>3:05:00) groups

3. DESCRIPTIVES.ipynb
   Reads:  cohort_success.csv, cohort_failure.csv
   Action: Generate Table 1 (demographics) and Table 2 (segment paces)

4. CLUSTERING.ipynb
   Reads:  cohort_success.csv, cohort_failure.csv
   Writes: cohort_clustered.csv (9,585 runners with cluster assignment)
   Action: K-Means clustering (k=2), silhouette validation

5. PREDICTION.ipynb
   Reads:  cohort_clustered.csv
   Action: Logistic regression, Random Forest, ROC analysis, CV threshold

6. FIGURES.ipynb
   Reads:  cohort_success.csv, cohort_failure.csv, cohort_clustered.csv
   Writes: ../figures/Figure_1–5.tiff + supplementary/Supp_1–2.tiff
   Action: Generate all publication figures (TIFF 600 DPI)
```

## File descriptions

| File | Records | Description |
|------|---------|-------------|
| `Dataset_Berlin_Marathon_1999-2025_original.csv` | 880,779 | Raw data (semicolon-separated, download from Zenodo) |
| `berlin_marathon_clean.parquet` | 834,168 | Cleaned and filtered (generated by CLEANING) |
| `cohort_success.csv` | 7,710 | Success group: finish 2:58:00–2:59:59 (generated by COHORT_DEFINITION) |
| `cohort_failure.csv` | 1,875 | Failure group: same HM pace, finish >3:05:00 (generated by COHORT_DEFINITION) |
| `cohort_clustered.csv` | 9,585 | Full cohort with cluster labels (generated by CLUSTERING) |

All files except the original CSV are generated by the pipeline and do not need to be downloaded separately.
